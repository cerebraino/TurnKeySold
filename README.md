## File Type Reference

### Lead Lists (`leads_[Domain].md`)
Each lead file contains:
- **Domain valuation**: comparable sales, brandability score (1–50), value range, anchor price
- **Industry targets**: primary and secondary buyer categories with buyer personas
- **Lead table**: 10–15 ranked leads with DMPS scores (Domain Match Potential Score, 0–100), company name, decision-maker name/role, website, and intent signals
- **Bundle strategies**: where applicable, cross-domain bundling recommendations
- **Google Search enrichment** (13 priority domains): top 5 paid advertisers + top 5 organic results for the domain's core keyword, with market-specific research

### Outreach Briefs (`brief_[Domain].md`)
Each brief contains:
- **Domain overview**: niche, target market, value proposition
- **Lead research summary**: top 5 leads from the research file
- **Messaging strategy**: primary angle, key talking points, tone guidance
- **Email templates**: 2–3 email variants (bilingual where applicable)
- **LinkedIn approach**: connection request + follow-up scripts
- **Objection handling**: 3–5 common objections with responses
- **Negotiation**: internal pricing reference (floor, anchor, ideal)
- **All pricing hidden behind `> **Internal Reference Only**` blocks** — zero dollar figures in outreach copy

### Buyer Expansions (`*-buyers.md`, `*-expansion.md`)
Strategic deep-dives on specific domain clusters:
- 40+ buyer companies per expansion
- Categorized by vertical (brewers/retailers/fashion/media/fintech, etc.)
- Named executives with LinkedIn profiles where public
- DMPS scores and acquisition rationale per company
- Pitch templates and outreach sequences

### Google Alerts (`google_alerts_v2.csv`)
CSV with columns: `Domain, Value Range, Primary Query, Secondary Query, Market`
- 144 rows covering all 127 researched domains
- Each query uses buyer-signal keywords ("funding", "launch", "acquires", "raises", "expands into")
- OR operators with exact quoted phrases
- Market-specific (US, ES, FR, PT, MX, LATAM, Global)

### Corporate Intelligence (`*-intel.md`)
Deep corporate profiles on known domain buyers:
- Entity structure, funding, leadership, FDA/regulatory history
- Tech stack, online properties, partnerships
- CEO identification roadmap
- Outreach strategy and draft pitches

## Google Alerts Setup

[`google_alerts_v2.csv`](google_alerts_v2.csv) contains 144 rows — each domain gets 2–3 queries. Set up at [google.com/alerts](https://www.google.com/alerts) with:
- **Frequency**: As-it-happens
- **Sources**: Automatic
- **Language**: Match the domain's market (Spanish for ES/LATAM, French for FR, Portuguese for PT/BR)
- **Region**: Match the domain's primary market
- **Deliver to**: Your preferred email

Priority domains to set up first (highest value): TopSex.ai, LaVoiture.ai, WhoreHouse.ai, buyslimmeds.com, bestslimmeds.com, automoviles.ai, NoBreak.ai, KnowLaw.ai, ChelaFria.com, MuyGuay.com.

## How to Use This Repository for Outreach

1. **Pick a domain** → navigate to `DOMAINS/[Domain]/`
2. **Read the lead list** → `01-research/leads_[Domain].md` — identify top 5 targets by DMPS
3. **Read the brief** → `02-outreach/brief_[Domain].md` — copy email/LinkedIn templates
4. **Check for expansions** → look for `*-buyers.md` or `*-expansion.md` in the research folder
5. **Personalize** → replace [Name], [Company Name] placeholders with actual decision-maker details from the lead table
6. **Send** → use pricing-free templates. Only share pricing if the buyer expresses interest.
7. **Monitor** → Google Alerts from `google_alerts_v2.csv` will surface new buyer signals (funding rounds, product launches, acquisitions)

## DMPS Scoring System

**Domain Match Potential Score (0–100)** measures how likely a company is to buy the domain:

| Range | Tier | Description |
|---|---|---|
| 90–100 | Hot | Direct category fit, high intent signals, budget likely |
| 70–89 | Warm | Adjacent category, growing into space, moderate signals |
| 50–69 | Lukewarm | Peripheral fit, speculative play, low signals |
| <50 | Cold | Remote fit, unlikely buyer without trigger event |

**+10 DMPS bonus**: Companies already running paid ads on the domain's core keyword (from Google Search enrichment).
